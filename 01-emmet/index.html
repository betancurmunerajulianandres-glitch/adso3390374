01-Emmet

Pluglin for faster HTML and CSS coding.

01-child

creates child elements (nesting)

Example

ul>li>a

Result

<ul> <li> <a href=""></a> </li> </ul>

02-Sibling +

Creates sibling elements (at the same level)

Example:

section+header+nav

Result:

<section></section> <header></header> <nav></nav>

03-Repeat *

Repeats or multiplies elements

Example:

ul>li*3

Result:

<ul> <li></li> <li></li> <li></li> </ul>

04-Class Attribute .

Creates an element with a class name

Example:

div.contenedor

Result:

<div class="contenedor"></div>

05-Multiple Classes..

Creates an element with multiple classes

Example:

div.clase1.clase2.clasen

Result:

<div class="clase1 clase2 clasen"></div>

06-ID Attribute #

Creates an element with an ID name

Example:

nav#menu

Result:

<nav id="menu"></nav>

07-Add Text {}

Adds text content inside an element

Example:

div{Lorem Ipsum Dolor}

a{Click Aquí}

Result:

<div>Lorem Ipsum Dolor</div> <a href="">Click Aquí</a>

08-Custom Attributes []

Defines custom attributes for an element

Example:

a[href=www.google.com]

Result:

<a href="www.google.com"></a>

09-Item Numbering $

Generates consecutive numbering for class names, IDs, or content

Example:

ul>li.num$*3

Result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

10-Item Numbering Direction @-

Changes the direction of consecutive numbering (descending)

Example:

ul>li.num$@-*3

Result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

11-Item Numbering Base @n

Defines the starting base number for consecutive numbering

Example:

ul>li.num$@3*4

Result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

12-Climb-up ^

Climbs up one level in the element tree hierarchy

Example:

section>article>div^p

Result:

<section> <article> <div></div> </article> <p></p> </section>

13-Grouping ( )

Groups elements to define complex markup structures

Example:

(header>nav)(section>article)

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14-Lorem Ipsum lorem

Generates placeholder "Lorem Ipsum" text

Example:

lorem

lorem20

lorem*5

15-Documents HTML html:

Generates complete HTML document structures

Example:

html:xs — Documento xhtml 1.0 Strict

html:xt — Documento xhtml 1.0 Transitional

html:4s — Documento html 4.01 Strict

html:4t — Documento html 4.01 Transitional

html:5 — Documento html 5

16-Meta TAGS meta:

Generates meta tags quickly

Example:

meta:vp — <meta name="viewport" content="width=device-width...">
